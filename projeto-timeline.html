Timeline - Site Baseado em Design Figma

Sobre o Projeto

Este projeto foi desenvolvido a partir de um design criado no Figma, onde o desafio foi transformar um design estático em um site totalmente funcional e responsivo. O site apresenta uma timeline visual que exibe informações de forma organizada e atraente, seguindo fielmente o design original.

O objetivo principal foi demonstrar habilidades em converter designs em código, mantendo a fidelidade visual e garantindo uma experiência de usuário excelente. Todas as animações, cores e espaçamentos foram implementados seguindo as especificações do design.

Objetivo

Este projeto foi criado com os seguintes objetivos:

  • Converter design do Figma em código HTML/CSS funcional
  • Criar uma interface visualmente atraente e moderna
  • Implementar uma timeline interativa e responsiva
  • Garantir fidelidade ao design original
  • Demonstrar habilidades em desenvolvimento front-end
  • Criar uma experiência de usuário fluida e agradável

Tecnologias Utilizadas

HTML5

Estrutura semântica

CSS3

Estilização e animações

O Que Foi Utilizado

HTML5: Utilizei HTML5 semântico para estruturar o conteúdo do site, garantindo uma base sólida e acessível. A estrutura foi pensada para ser clara, organizada e fácil de manter.

CSS3: Com CSS3, implementei todos os estilos visuais, incluindo layouts flexíveis, animações suaves, gradientes e efeitos de transição. O CSS foi usado para recriar fielmente o design do Figma, incluindo cores, espaçamentos, tipografia e todos os detalhes visuais.

O projeto demonstra habilidades em desenvolvimento front-end puro, sem dependência de frameworks, focando em código limpo e bem estruturado que é fácil de entender e manter.
