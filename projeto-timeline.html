Timeline

Sobre o Projeto

Projeto desenvolvido a partir de um design criado no Figma, com o desafio de transformar um design estático em um site totalmente funcional e responsivo. O site apresenta uma timeline visual que exibe informações de forma organizada e atraente, seguindo fielmente o design original.

O objetivo principal foi demonstrar habilidades em converter designs em código, mantendo a fidelidade visual e garantindo uma experiência de usuário excelente. Todas as animações, cores e espaçamentos foram implementados seguindo as especificações do design.

Objetivos

  • Converter design do Figma em código HTML/CSS funcional
  • Criar uma interface visualmente atraente e moderna
  • Implementar uma timeline interativa e responsiva
  • Garantir fidelidade ao design original
  • Demonstrar habilidades em desenvolvimento front-end puro
  • Criar experiência de usuário fluida e agradável

Tecnologias

HTML5Estrutura semântica
CSS3Animações e layout

Como foi feito

HTML5: Estrutura semântica clara, organizada e fácil de manter, garantindo base sólida e acessível para o site.

CSS3: Todos os estilos visuais, layouts flexíveis, animações suaves, gradientes e efeitos de transição para recriar fielmente o design do Figma. Cores, espaçamentos e tipografia seguem as especificações originais.

O projeto demonstra habilidades em desenvolvimento front-end puro, sem dependência de frameworks, com foco em código limpo e bem estruturado.
