Pix Premiado - Tio Léo Utilidades

Sobre o Projeto

O Pix Premiado é um sistema web completo desenvolvido para gerenciar campanhas promocionais tipo rifa com 40 números. O sistema foi criado para a Tio Léo Utilidades e permite que os clientes reservem números, realizem pagamentos via Pix e acompanhem o status de suas reservas em tempo real.

O projeto oferece uma interface responsiva e intuitiva tanto para os participantes quanto para os administradores, com funcionalidades como reserva temporária de 15 minutos, confirmação manual de pagamentos, painel administrativo completo e integração com WhatsApp para envio de comprovantes.

Objetivo

Este projeto foi desenvolvido com os seguintes objetivos:

  • Criar um sistema completo para gerenciar campanhas promocionais tipo Pix Premiado (rifa com 40 números)
  • Implementar sistema de reserva em 2 etapas: nome/telefone obrigatórios e dados do Pix
  • Desenvolver reserva temporária de 15 minutos com liberação automática de reservas expiradas
  • Criar sistema de status para números: Disponível, Reservado, Aguardando Confirmação e Pago
  • Desenvolver painel administrativo com login por senha para gerenciar pagamentos e números
  • Implementar atualização em tempo real do grid de números na página inicial
  • Integrar link para WhatsApp para envio de comprovante após pagamento
  • Garantir interface responsiva para desktop, tablet e celular
  • Implementar segurança com validações no backend, prepared statements e proteção contra SQL injection

O Que Foi Utilizado

HTML5: Utilizei HTML5 semântico para estruturar o conteúdo do sistema de forma organizada e acessível. A estrutura foi pensada para facilitar a navegação e melhorar a experiência do usuário tanto na página inicial quanto no painel administrativo.

CSS3: Com CSS3, criei um design moderno e responsivo que funciona perfeitamente em desktop, tablet e celular. Implementei layouts flexíveis, animações suaves, gradientes e efeitos visuais que tornam a experiência do usuário agradável e profissional. O sistema utiliza variáveis CSS para personalização de cores.

JavaScript: Implementei JavaScript para adicionar interatividade ao sistema, incluindo funcionalidades como atualização em tempo real do grid de números via AJAX, validação de formulários no lado do cliente, manipulação do DOM para exibir modais de reserva e criação de uma experiência de usuário fluida e responsiva.

PHP: Utilizei PHP como linguagem de programação backend para desenvolver a lógica do servidor, criar APIs RESTful para comunicação com o frontend, gerenciar sessões de administrador, realizar operações no banco de dados e implementar todas as funcionalidades do sistema, incluindo validações de segurança e proteção contra SQL injection usando prepared statements.

MySQL: Escolhi MySQL como sistema de gerenciamento de banco de dados relacional para armazenar todas as informações do sistema, incluindo dados dos números (status, nome, telefone, datas de reserva e pagamento). O MySQL proporciona uma base sólida e confiável para aplicações web que requerem armazenamento estruturado de dados.

O sistema foi desenvolvido com foco em segurança, usabilidade e desempenho, garantindo que tanto os participantes quanto os administradores tenham uma experiência positiva e eficiente ao utilizar a plataforma.

Funcionalidades Principais

  • Página Inicial: Exibição do prêmio, regras e grid de 40 números com atualização em tempo real
  • Reserva em 2 Etapas: Nome e telefone obrigatórios na primeira etapa, dados do Pix e botão "Já paguei" na segunda
  • Reserva Temporária: 15 minutos de validade com liberação automática de reservas expiradas
  • Sistema de Status: Disponível, Reservado, Aguardando Confirmação e Pago
  • Painel Admin: Login por senha, confirmar pagamento, liberar número, editar nome/telefone e resetar todos os números
  • Integração WhatsApp: Link para envio de comprovante após "Já paguei"
  • Interface Responsiva: Funciona perfeitamente em desktop, tablet e celular
