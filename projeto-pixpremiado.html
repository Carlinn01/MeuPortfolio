Pix Premiado

Sistema Web Full Stack PHP MySQL JavaScript Tio Léo Utilidades

Sobre o Projeto

Sistema web completo para gerenciar campanhas promocionais tipo rifa com 40 números. Criado para a Tio Léo Utilidades, permite que clientes reservem números, realizem pagamentos via Pix e acompanhem o status das reservas em tempo real.

Interface responsiva para participantes e administradores, com reserva temporária de 15 minutos, confirmação manual de pagamentos, painel administrativo completo e integração com WhatsApp para envio de comprovantes.

Objetivos

  • Gerenciar campanhas promocionais tipo Pix Premiado (rifa com 40 números)
  • Sistema de reserva em 2 etapas: dados do cliente e dados do Pix
  • Reserva temporária de 15 minutos com liberação automática
  • Sistema de status: Disponível, Reservado, Aguardando Confirmação e Pago
  • Painel admin com login por senha para gerenciar pagamentos
  • Atualização em tempo real do grid de números
  • Integração WhatsApp para envio de comprovante
  • Segurança com prepared statements e proteção contra SQL injection

Tecnologias

Funcionalidades principais

  • Página Inicial: grid de 40 números com atualização em tempo real
  • Reserva em 2 Etapas: nome/telefone → dados do Pix + "Já paguei"
  • Reserva Temporária: 15 minutos de validade com liberação automática
  • Painel Admin: confirmar pagamento, liberar número, editar dados, resetar tudo
  • WhatsApp: link para envio de comprovante após pagamento
  • Segurança: prepared statements, validação no backend, proteção SQL injection
